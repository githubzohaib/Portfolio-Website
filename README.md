# Portfolio Website

A personal developer portfolio built using the **MERN Stack** to showcase my projects, professional experience, certifications, and technical skills. The website provides an interactive interface where visitors can explore my work and contact me.

## Pages

### Home

Introduces my profile, highlights my skills, and provides quick navigation to other sections.

### About

Brief overview of my background, interests, and development focus.

### Experience

Displays my professional experience and development journey.

### Projects

Showcases selected projects with details about the technologies used and the purpose of each project.

### Internship

Lists professional Internships Experience.

### Hackathon

Lists Hackathon Experience.

### Contact

Allows visitors to reach me through contact information and a contact form.

## Features

* Responsive modern UI
* Smooth navigation between sections
* Interactive animations
* Project showcase
* Experience timeline
* Certificates display
* Contact form with backend handling

## Tech Stack

**Frontend**

* React
* Tailwind CSS
* Framer Motion

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB
