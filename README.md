## 📂 Featured Projects

### 1. [ServiceHub](https://frontend-pro-battle26-stage3.vercel.app/login)
Full-stack service marketplace connecting seekers and providers with secure three-role authentication (Admin, Seeker, Provider), real-time WebSocket chat, and H3 geospatial indexing for radius-based worker discovery.
`React.js` `Node.js` `Express.js` `PostgreSQL` `WebSockets` `H3`

### 2. [Student Score Predictor](https://students-prediction-model.vercel.app/)
ML-powered academic performance predictor using study hours, attendance, and classroom participation as features, with a React + Tailwind frontend and FastAPI backend for real-time predictions.
`React.js` `FastAPI` `Tailwind CSS` `Machine Learning`

### 3. [HelpGhar](https://help-ghar-beta.vercel.app/)
Full-stack home services marketplace connecting customers with verified workers. Includes a custom RAG-based chatbot (vector embeddings) for answering user queries, and a scikit-learn model for gig ranking and personalized worker recommendations.
`React.js` `TypeScript` `RAG` `Machine Learning`
Frontend deployed on Vercel, backend on Render.
